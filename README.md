# Il modello tra problema e soluzione

In questo repository sono disponibili i materiali del corso _Il modello tra problema e soluzione_ tenuto nell'anno scolastico 2024/2025 presso l'Istituto d'Istruzione Superiore "Savoia Benincasa" di Ancona (AN).

Il corso propone la centralità del modello nel processo di soluzione dei problemi, evidenziando come la capacità di descrivere un problema in modo formale possa permettere di risolverlo anche senza implementare un algoritmo.
I problemi affrontati nel corso sono problemi decisionali e la modellizzazione adottata è quella della programmazione matematica.

La presentazione illustra principalmente la metodologia didattica impiegata, che descrive il processo di ricerca di soluzioni a problemi complessi che richiedono risposte quantitative.
